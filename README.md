# Wizard Rogue-like

A tiny top-down roguelite runner in a single HTML file — vanilla JS + Canvas, no dependencies.

**Play it:** https://tonka13.github.io/wizard-rogue-like/

## Controls

- **WASD / Arrow keys** — move
- **Mouse** — aim
- **Click / hold** — shoot
- **Esc** — pause

## The loop

Survive scrolling waves of imps and bats, beat the boss at each checkpoint, spend gems on upgrades (regen shield, multi-attack, extra life), and push into ever-harder segments. Death resets everything — see your stats and go again.
